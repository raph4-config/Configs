# Neovim / LazyVim

Leader = `Space`. Press `Space` alone to see the which-key menu (discover everything).

## Move fast (no arrows)
| Move | Key |
|------|-----|
| Word fwd / back / end | `w` / `b` / `e` |
| Line start / first char / end | `0` / `^` / `$` |
| File top / bottom | `gg` / `G` |
| Half page up / down | `‚åÉ + u` / `‚åÉ + d` |
| Jump to char in line | `f<char>` (`;` repeats) |
| Matching bracket | `%` |
| Paragraph up / down | `{` / `}` |
| Back / forward in jumps | `‚åÉ + o` / `‚åÉ + i` |
