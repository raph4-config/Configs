# macOS

`⌘` Cmd · `⌥` Option · `⌃` Ctrl · `⇧` Shift

## Text navigation (kill the arrow-spam)
| Move | Key |
|------|-----|
| Word left / right | `⌥ + ← / →` |
| Line start / end | `⌘ + ← / →` |
| Doc top / bottom | `⌘ + ↑ / ↓` |
| Delete word back | `⌥ + ⌫` |
| Delete to line start | `⌘ + ⌫` |
| Select | add `⇧` to any move above |

## Windows & apps
| Action | Key |
|--------|-----|
| Switch app | `⌘ + Tab` |
| Switch windows of app | `⌘ + ` (backtick) |
| Spotlight (launch/search) | `⌘ + Space` |
| Hide app / Minimize | `⌘ + H` / `⌘ + M` |
| Close window / Quit app | `⌘ + W` / `⌘ + Q` |
| Fullscreen | `⌃ + ⌘ + F` |

## Spaces / Mission Control
| Action | Key |
|--------|-----|
| Mission Control | `⌃ + ↑` |
| Switch desktop | `⌃ + ← / →` |
| App windows (Exposé) | `⌃ + ↓` |

## Screenshots
| Action | Key |
|--------|-----|
| Selection to file | `⌘ + ⇧ + 4` |
| Capture menu (record, options) | `⌘ + ⇧ + 5` |
| Selection to clipboard | add `⌃` |

## Finder
| Action | Key |
|--------|-----|
| Go to folder | `⌘ + ⇧ + G` |
| Up / Open | `⌘ + ↑` / `⌘ + ↓` |
| New folder | `⌘ + ⇧ + N` |
